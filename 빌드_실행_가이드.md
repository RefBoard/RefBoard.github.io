# Windows 설치 파일 빌드 실행 가이드

## 🚀 빠른 시작

### 한 번에 빌드하기

CMD에서 다음 명령어를 실행하세요:

```bash
cd D:\Scripts\RefBoard
npm run electron:build
```

이 명령어는 자동으로:
1. TypeScript 컴파일
2. React 앱 빌드 (`dist/` 폴더 생성)
3. Electron 앱 패키징 (`release/` 폴더에 설치 파일 생성)

---

## 📦 빌드 결과물

빌드가 완료되면 `release/` 폴더에 다음 파일이 생성됩니다:

- **`RefBoard Setup 0.0.0.exe`** ← 이것이 설치 파일입니다!

이 파일을 사용자에게 배포하면 됩니다.

---

## ✅ 빌드 전 확인사항

1. **의존성 설치 확인**
   ```bash
   npm install
   ```

2. **개발 모드로 테스트 (선택사항)**
   ```bash
   npm run electron:dev
   ```
   - 앱이 정상적으로 실행되는지 확인
   - 기능이 모두 작동하는지 테스트

---

## 🔧 단계별 빌드 (선택사항)

### 1단계: React 앱 빌드

```bash
npm run build
```

**결과:** `dist/` 폴더에 빌드된 파일 생성

### 2단계: Electron 패키징

```bash
electron-builder --win
```

**결과:** `release/` 폴더에 설치 파일 생성

- **설치 프로그램 타입:** NSIS (Windows 표준 설치 프로그램)

## 🐛 문제 해결

### 빌드가 실패하는 경우

1. **에러 메시지 확인**
   - 콘솔에 표시된 에러 메시지를 확인하세요

2. **의존성 재설치**
   ```bash
   npm install
   ```

3. **캐시 삭제 후 재빌드**
   ```bash
   npm run build -- --force
   npm run electron:build
   ```

### 빌드는 성공했지만 앱이 실행되지 않는 경우

1. **개발 모드로 테스트**
   ```bash
   npm run electron:dev
   ```

2. **콘솔 에러 확인**
   - 개발 모드에서 DevTools가 자동으로 열립니다
   - Console 탭에서 에러 확인

## ✅ 완료 후 확인

빌드가 성공하면:

1. **`release/` 폴더 확인**
   - `RefBoard Setup 0.0.0.exe` 파일이 있는지 확인

2. **설치 파일 테스트**
   - `.exe` 파일을 더블클릭하여 설치 테스트
   - 설치된 앱이 정상적으로 실행되는지 확인

3. **기능 테스트**
   - 로그인 기능
   - 보드 생성/편집
   - 이미지/동영상 업로드
   - PSD 파일 업로드
