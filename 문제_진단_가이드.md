# 문제 진단 가이드

## 🔍 현재 상황
접속해도 아무것도 뜨지 않습니다.

---

## ✅ 1단계: 브라우저 콘솔 확인 (가장 중요!)

1. **브라우저에서 사이트 접속**
   - `https://trackk81.github.io/refboard-website/refboard/` 접속

2. **개발자 도구 열기**
   - `F12` 키 누르기
   - 또는 마우스 우클릭 → "검사" 클릭

3. **Console 탭 확인**
   - 빨간색 에러 메시지가 있는지 확인
   - 에러 메시지를 복사해서 알려주세요

4. **Network 탭 확인**
   - 파일들이 제대로 로드되는지 확인
   - 빨간색(실패) 항목이 있는지 확인

---

## ✅ 2단계: 배포된 파일 확인

GitHub에서 배포된 파일을 확인하세요:

1. **리포지토리 접속**
   - `https://github.com/trackk81/refboard-website` 접속

2. **refboard 폴더 확인**
   - `refboard` 폴더 클릭
   - `index.html` 파일이 있는지 확인
   - `assets` 폴더가 있는지 확인

3. **index.html 내용 확인**
   - `index.html` 파일 클릭
   - "Raw" 버튼 클릭
   - 내용이 올바른지 확인

---

## ✅ 3단계: 다시 빌드 및 배포 확인

### 다시 빌드했는지 확인

**중요:** `vite.config.web.ts`를 수정한 후 **반드시 다시 빌드**해야 합니다!

```bash
cd D:\Scripts\RefBoard
npm run build:web
```

### 파일 복사 확인

1. `D:\Scripts\RefBoard\dist-web\index.html` 파일 열기
2. 내용 확인:
   ```html
   <script type="module" crossorigin src="/refboard/assets/index-B0p4sZXv.js"></script>
   <link rel="stylesheet" crossorigin href="/refboard/assets/index-Blo1H1iY.css">
   ```
   - 경로가 `/refboard/assets/...`로 시작하는지 확인

3. `D:\Scripts\refboard-website\refboard\index.html` 파일 열기
4. 두 파일이 동일한지 확인

---

## ✅ 4단계: GitHub Pages 설정 확인

1. **리포지토리 Settings**
   - `https://github.com/trackk81/refboard-website/settings/pages` 접속

2. **설정 확인**
   - Source: "Deploy from a branch"
   - Branch: "main"
   - Folder: "/ (root)"

3. **배포 상태 확인**
   - "Pages" 섹션에서 최근 배포 확인
   - "Your site is live at..." 메시지 확인

## 🐛 일반적인 문제와 해결책

### 문제 1: 빈 페이지 (하얀 화면)

**원인:**
- JavaScript 파일이 로드되지 않음
- 경로 문제

**해결책:**
- 브라우저 콘솔에서 에러 확인
- `index.html`의 스크립트 경로 확인
- `vite.config.web.ts`의 `base` 설정 확인

### 문제 2: 404 에러

**원인:**
- 파일이 GitHub에 업로드되지 않음
- 경로가 잘못됨

**해결책:**
- GitHub 리포지토리에서 파일 확인
- `git push`가 성공했는지 확인

### 문제 3: 자산 로드 실패

**원인:**
- `base` 설정이 잘못됨
- 파일 경로가 상대 경로가 아님

**해결책:**
- `vite.config.web.ts`의 `base`를 `/refboard/`로 설정
- 다시 빌드
- 파일 다시 복사

### 문제 4: Firebase/Google 인증 에러

**원인:**
- 환경 변수 설정 문제
- Firebase 설정 문제

**해결책:**
- 브라우저 콘솔에서 에러 확인
- Firebase 프로젝트 설정 확인
