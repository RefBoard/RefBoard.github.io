RefBoard 사용자 가이드

RefBoard를 200% 활용하는 방법을 상세히 안내해 드립니다. 이 가이드를 통해 기본적인 사용법부터 고급 기능까지 마스터하여 창작 프로세스를 혁신하세요.

✨ 시작하기

RefBoard는 단순한 이미지 뷰어가 아닙니다. 무한한 캔버스 공간에서 수천 장의 레퍼런스를 정리하고, 팀원과 실시간으로 아이디어를 발전시키는 통합 비주얼 워크스페이스입니다.

설치 및 실행:
Windows 사용자는 [다운로드] 버튼을 통해 설치 파일을 받으실 수 있습니다. 설치 후 Google 계정으로 로그인하면 즉시 Google Drive와 연동되어 클라우드 저장이 활성화됩니다.

🚀 주요 기능 마스터하기

1. 무한 캔버스 활용

RefBoard의 캔버스는 크기 제한이 없습니다. 화면의 빈 공간을 드래그하여(Pan) 자유롭게 이동하고, 마우스 휠(또는 Ctrl + 휠)로 부드럽게 줌인/줌아웃 할 수 있습니다. 프로젝트의 규모가 아무리 커져도 하나의 보드에 모든 자료를 담을 수 있습니다.

2. 미디어 가져오기 (Drag & Drop)

가장 빠르고 강력한 기능입니다. 바탕화면의 이미지 파일이나 폴더, 또는 웹 브라우저의 이미지를 RefBoard 창으로 드래그 앤 드롭하세요.

  • 이미지(JPG, PNG, GIF, WEBP): 즉시 로드되어 캔버스에 배치됩니다.
  • 비디오(MP4): 자동 재생/루프 설정이 가능한 플레이어로 삽입됩니다.
  • PSD 파일: Photoshop 없이도 레이어 미리보기가 가능한 고화질 이미지로 변환되어 표시됩니다.

3. 스마트 그룹 & 정렬

흩어진 이미지를 선택하고 Ctrl + G를 누르면 그룹이 생성됩니다. 그룹 내에서 이미지들은 자동으로 격자(Grid) 형태로 정렬되며, 그룹 제목을 더블 클릭하여 이름을 변경할 수 있습니다 ("캐릭터 시안", "배경 레퍼런스" 등).

🎨 AI 이미지 생성 (Gemini Pro)

아이디어가 막혔을 때, RefBoard에 내장된 Gemini Pro AI를 활용하세요. 우클릭 메뉴에서 "AI 이미지 생성"을 선택하거나 도구 모음의 AI 아이콘을 클릭합니다.

프롬프트를 입력하면 고화질의 레퍼런스 이미지가 생성되어 즉시 캔버스에 추가됩니다. 별도의 브라우저를 열 필요 없이, 작업 흐름을 끊지 않고 영감을 얻을 수 있습니다.

🤝 실시간 협업 가이드

원격 근무 시대의 필수 기능입니다. 보드 우측 상단의 [공유] 버튼을 눌러 팀원을 초대하세요.

  • 실시간 커서: 상대방이 보고 있는 위치와 마우스 움직임이 이름표와 함께 실시간으로 표시됩니다.
  • 동시 편집: 누군가 이미지를 옮기거나 크기를 조절하면, 모든 팀원의 화면에 즉시 반영됩니다.
  • 드로잉 피드백: 펜 도구(P)를 사용하여 이미지 위에 직접 수정 사항을 그리거나 메모를 남기세요.

⌨️ 필수 단축키 모음

  • Space + 드래그 : 화면 이동 (Pan)
  • Ctrl + Z : 실행 취소
  • Ctrl + Y : 다시 실행
  • Del : 선택한 아이템 삭제
  • Ctrl + C / V : 복사 / 붙여넣기
  • Ctrl + G : 그룹 만들기
  • Ctrl + Shift + G : 그룹 해제
  • Alt + 드래그 : 아이템 복제 이동

❓ 자주 묻는 질문 (FAQ)

Q. RefBoard는 무료인가요?
A. 네, 현재 모든 핵심 기능은 무료로 제공됩니다. 개인 사용자와 소규모 팀은 비용 부담 없이 무제한으로 보드를 생성할 수 있습니다.

Q. 데이터는 어디에 저장되나요?
A. 모든 데이터는 사용자의 Google Drive에 저장되므로 보안이 강력하며, 저희 서버에는 파일을 저장하지 않아 프라이버시가 완벽하게 보호됩니다.

Q. Google 계정이 꼭 필요한가요?
A. 네, RefBoard는 Google Drive를 저장소로 사용하기 때문에 Google 계정 로그인이 필요합니다. 덕분에 별도의 회원가입 없이 기존 계정으로 바로 시작할 수 있습니다.
