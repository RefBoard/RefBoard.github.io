# 스크린샷 이미지 준비 가이드

## 이미지 준비 방법

1. **앱 스크린샷 촬영**
   - RefBoard 앱을 실행하고 가장 대표적인 화면을 캡처하세요
   - 권장 해상도: 1800x1000px 이상 (비율 16:9 또는 9:16)
   - 파일 형식: PNG 또는 JPG

2. **이미지 파일 이름**
   - `app-screenshot.png` 또는 `app-screenshot.jpg`
   - 이 폴더(`images/`)에 저장하세요

3. **이미지 최적화 (선택사항)**
   - 파일 크기를 줄이기 위해 이미지 압축 도구 사용 권장
   - 온라인 도구: TinyPNG, Squoosh 등

4. **HTML에 이미지 추가**
   - `index.html` 파일의 52번째 줄 근처에서 주석을 해제하고 이미지 경로를 확인하세요:
   ```html
   <img src="images/app-screenshot.png" alt="RefBoard 앱 스크린샷" />
   ```

## 추천 스크린샷 화면

- 메인 캔버스 화면 (여러 이미지가 배치된 상태)
- AI 이미지 생성 노드가 보이는 화면
- 실시간 협업 화면
- PSD 파일이 열려있는 화면
